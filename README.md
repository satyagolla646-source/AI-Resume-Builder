# AI-Resume-Builder
An AI-powered resume builder using Flask, Gemini AI, and PDF generation.
## Features
- User-friendly resume form
- AI-generated professional summary
- PDF resume download
- Responsive UI

## Technologies Used
- Python
- Flask
- Gemini AI
- HTML
- CSS
- ReportLab

## Run Locally

pip install -r requirements.txt
python app.py
